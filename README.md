# 🇧🇩 Bangladesh Geo Data - Complete Divisions, Districts & Upazilas

[![npm version](https://img.shields.io/npm/v/bangladesh-geo-data.svg)](https://www.npmjs.com/package/bangladesh-geo-data)
[![npm downloads](https://img.shields.io/npm/dm/bangladesh-geo-data.svg)](https://www.npmjs.com/package/bangladesh-geo-data)
[![license](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

**bangladesh-geo-data** is a lightweight npm package providing complete geographical data of Bangladesh, including divisions, districts, and upazilas (sub-districts) with both English and Bengali names. Perfect for location selectors, forms, address systems, and Bangladeshi applications.

---

## 🚀 Features

- **8 Divisions** with English and Bengali names
- **64+ Districts** mapped to their respective divisions
- **500+ Upazilas** (sub-districts) mapped to districts
- **Full TypeScript support** with complete type definitions
- **Utility functions** for easy data retrieval
- **Hierarchical data structure** (Division → District → Upazila)
- **Zero dependencies** - lightweight and fast
- **Works in Node.js, React, Next.js, and browser**
